Here's a few of my recent personal projects from hackathons

Parrrley

Parrley is an English-to-Piratespeak translator created by a team of four in 24 hours for MLH's HackItShipIt Hackathon. Parrley not only translates, but also measures the "pirate-ness" of a user's input before and after translation.

Tech: HTML, CSS, Reactstrap, Javascript, React + Hooks, Git/Github

  • Check it out on Github

CodeXplode

CodeXplode is a fun block-matching game that guides young kids and beginners through language-agnostic coding syntax. This app was created by a group of three in one day for MLH's Hacky Birthday Hackathon.

Tech: HTML, CSS, React-Bootstrap, Javascript, React + Hooks, Git/Github

  • Check it out on Github

Shipwrecked

Shipwrecked is a map of shipwrecks that have happened all over the world. Click on any ship icon to read more about the wreck itself. You may also dive further into the wreck to learn more details about the ship and its watery fate.

Tech: HTML, Javascript, React + Hooks, Git/Github, Google Maps API, Wikipedia API

  • Check it out on Github
